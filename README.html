Literate programming avec Org mode

Pour information, ce document Org est disponible sur https://github.com/fniessen/gutenberg.

Org mode est une « fonctionnalité » d’Emacs offrant (entre autres) une syntaxe plain text facile à utiliser pour exporter des documents en LaTeX, en HTML, en documents LibreOffice – et encore plein d’autres choses !

2 Exécution de code

Voici un bout de code Python (plus-2-fois-2) qui ajoute 2, puis multiplie par 2; d’où son nom : 

resultat = x + 2
resultat = resultat * 2
return resultat

30

Voici bout de code Emacs Lisp (fois-3) qui multiplie par 3, et dont seul le résultat m’intéresse : 

90

Ensuite, voici un bout de code en LaTeX qui génère différents résultats, pour quelques valeurs possibles de la variable x passées à plus-2-fois-2 : 

\begin{itemize}
\item 24
\item 44
\item 64
\end{itemize}

Et, finalement, un graphique produit en langage R. 

plot(1:10, (1:10)^2)

7 Annexes

7.1 Code Emacs Lisp

Code utilisé pour améliorer le rendu de la démonstration : 

(set-frame-font "Consolas-13" nil t)

(setq org-latex-default-packages-alist
      '(("AUTO" "inputenc" t
         ("pdflatex"))
        ("T1" "fontenc" t
         ("pdflatex"))
        ("" "graphicx" t)
        ("" "xcolor")
        ("" "babel")
        ("" "listings")
        ("" "hyperref" nil)))

(setq org-latex-packages-alist nil)

(setq org-latex-text-markup-alist '((bold . "\\textbf{%s}")
                                    (code . "\\lstinline|%s|")
                                    (italic . "\\emph{%s}")
                                    (strike-through . "\\sout{%s}")
                                    (underline . "\\uline{%s}")
                                    (verbatim . "\\verb|%s|")))

(setq org-latex-classes

      '(("article" "\\documentclass[11pt]{article}"
         ("\\section{%s}" . "\\section*{%s}")
         ("\\subsection{%s}" . "\\subsection*{%s}")
         ("\\subsubsection{%s}" . "\\subsubsection*{%s}")
         ("\\paragraph{%s}" . "\\paragraph*{%s}")
         ("\\subparagraph{%s}" . "\\subparagraph*{%s}"))

        ("report" "\\documentclass[11pt]{report}"
         ("\\chapter{%s}" . "\\chapter*{%s}")
         ("\\section{%s}" . "\\section*{%s}")
         ("\\subsection{%s}" . "\\subsection*{%s}")
         ("\\subsubsection{%s}" . "\\subsubsection*{%s}")
         ("\\paragraph{%s}" . "\\paragraph*{%s}"))

        ("book" "\\documentclass[11pt]{book}"
         ("\\chapter{%s}" . "\\chapter*{%s}")
         ("\\section{%s}" . "\\section*{%s}")
         ("\\subsection{%s}" . "\\subsection*{%s}")
         ("\\subsubsection{%s}" . "\\subsubsection*{%s}")
         ("\\paragraph{%s}" . "\\paragraph*{%s}"))))
